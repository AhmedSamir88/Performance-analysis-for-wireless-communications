This simulation code is mainly used to reproduce the results of the following paper [1]:

[1] Chongjun Ouyang, Yuanwei Liu, and Hongwen Yang, "On the Performance of Downlink MIMO-ISAC," submitted to IEEE Wireless Communications Letters, Sep. 2022

If you use this simulation code package in any way, please cite the original paper [1] above. The corresponding BiBTeX citation is given below:

@article{Ouyang2022SWCL,
  title={On the Performance of Downlink MIMO-ISAC},
  author={Ouyang, Chongjun and Liu, Yuanwei and Yang, Hongwen},
  year={2022},
  month=sep,
  publisher={https://arxiv.org/pdf/2209.01028.pdf}
}
The author in charge of this simulation code package is: Chongjun Ouyang (email: dragonaim@bupt.edu.cn).

Sensing_Rate.m: This is used to characterize the sensing rate in the paper
Rate_Region.m: This is used to characterize the rate region in the paper
Outage_Probability.m: This is used to characterize the outage probability of communication rate in the paper
Communication_Rate.m: This is used to characterize the ergodic communication rate in the paper

Note to change the parameters and modes according to your demand.
